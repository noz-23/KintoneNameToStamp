# KintoneNameToStamp

## Kintoneのユーザー選択をハンコ表示プラグイン

## 1.概要

Kintoneのユーザー選択表示をハンコの様にするプラグインです。

「名字のみ」「名字と名前1文字」「設定画像」を選択できます。

名字と名前はユーザー管理画面の値となります。

## 2.注意点

現状のハンコサイズは100px x 100 px 固定です。

表示位置は左揃えになります。
